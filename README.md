# Private Knowledge Q&A

A lightweight web application that allows users to upload private documents and ask questions that are answered **strictly based on the uploaded content**. The app also shows which document and snippet were used to generate the answer.

## ‚ú® Features

- Upload multiple text documents
- View list of uploaded documents
- Ask natural language questions
- Answers are generated only from uploaded documents
- Shows document source and supporting snippet
- System status page (backend, database, LLM)
- Basic validation for empty or invalid inputs
